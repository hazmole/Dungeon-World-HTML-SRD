地城世界 HTML SRD

地城世界是一款由 Sage Kobold Productions 公司旗下的 Sage LaTorra 和 Adam Koebel 所設計的桌上型角色扮演遊戲。這個網站是此遊戲的系統參考書（System Reference Document, SRD），以 HTML 的格式供瀏覽器閱覽。

本站借助了 Bootstrap 的設計，並透過 Hugo 建置而成。

若你想要為這個專案貢獻一份心力（比如說你發現了某個可以改良的地方、或是漏掉的翻譯、錯字等），請在 Github 專案上創建一個 pull request ，或提交到 原始專案上

《地城世界》的原始資料文件（衍生出這份 HTML SRD 文件的原檔）可在 GitHub 上找到： https://github.com/Sagelt/Dungeon-World
